# Beauty Rewards Landing

Vite + React + Tailwind landing page scaffolded for Netlify.

Quick start:

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

Deploy to Netlify: connect this repo to Netlify and set the build command to `npm run build` and publish directory to `dist`.
